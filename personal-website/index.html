🚀 Projects

🏧 OCBC ATM Prototype

A functional ATM prototype developed in 5 weeks for an OCBC problem statement. Designed to enhance user authentication, security, and accessibility, this project implemented cutting-edge banking technology.

🔹 Key Features:

  • 🔐 NFC & QR Code Logi – Enabled secure and seamless authentication.
  • 💸 Pre-Order Cash Feature – Allowed users to pre-order cash before reaching an ATM.
  • 🎨 Modern UI/UX – Designed an intuitive React-basedfrontend.
  • ⚙️ Secure Transaction – Developed backend logic for deposits, withdrawals, and authentication.

🛠️ Technologies Used:

  • Frontend: React.js, HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Security Features: NFC, QR Code Authentication
  • Database: MongoDB (NoSQL) for transaction logging

📌 Outcome:

  • ✅ Successfully presented at OCBC Campus in front of banking professionals.
  • ✅ Well-received prototype with future plans for AI-powered fraud detection.
